# Notepad

A simple Android Notepad app.

## Description

A minimal notepad application with a dark theme, no ads or network usage. just a notepad.

### Features
- write notes
- delete notes
- edit notes

## Getting Started

### Dependencies

*Android Studio (latest stable version recommended)
*Android SDK (API level XX or higher)
*Gradle (included with Android Studio)
*Android device or emulator running Android X.X+

### Installing
1. Clone the repository
2. Open the solution file in Android Studio
3. Build the project
